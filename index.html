What is the return on sales?
return on sales is a ratio used to evaluate a company's efficiency at converting it's sales into profit
How to calculate return on sales?
To calculate return on sales you should use the following formula:
return on sales = net income (before interest and tax) / sales
example: company x has an operating profit of $200,000 and it required sales worth $1,000,000 to generate it, calculate the return on sales of this company?
using the return on sales formula the answer will be:
reutrn on sales = net income (before interest and tax) / sales
return on sales = $200,000 / $1,000,000
return on sales = 20%

Why use return on sales?
companies use return on sales because it helps them determine what percentage of their sales is converted into profit.
